# notes-cli-app
Simple CLI note taking app in node.js

## How to use it
To use the app simply run it from the command line. Pass the help option for the commands available:
```
> node app.js --help

Commands:
  add     Add a new note
  read    Read a specific note
  list    List all notes
  remove  Remove a specific note

Options:
  --help  Show help  
```
